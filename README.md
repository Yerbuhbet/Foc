# Foc - Focus and Pomodoro App

Foc is an Android application designed to help users stay focused using the Pomodoro technique. It allows users to manage tasks and track their productivity.

## Features

- **Pomodoro Timer**: Customizable focus and break intervals.
- **Task Management**: Create, edit, and track tasks.
- **Local Storage**: Uses Room database for persistent task storage.
- **Modern UI**: Built with Jetpack Compose and Material 3.

## Tech Stack

- **Language**: Kotlin
- **UI Framework**: Jetpack Compose
- **Architecture**: MVVM
- **Database**: Room
- **Navigation**: Jetpack Navigation

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Yerbuhbet/Foc.git
   ```
2. Open the project in Android Studio.
3. Build and run the app on an emulator or a physical device.
